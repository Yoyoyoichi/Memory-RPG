# ROGUE-TEXT RPG - ゲーム基本仕様書

このドキュメントは、英語学習型テキストローグライクRPG（ROGUE-TEXT RPG）のゲームシステム、内部ロジック、UI構成、およびソースコード設計をまとめた仕様書です。Gemini等のAIモデルに読み込ませてアイデアを拡張するための基本リファレンスとして使用できます。

---

## 1. ゲーム概要
* **ジャンル:** 英語スペリング学習 × アスキーアート・ローグライクRPG
* **世界観:** レトロなサイバーグリッド・ダンジョン（アスキー文字で構成）
* **コアサイクル:** 
  1. キーボードまたは仮想D-padによるターン制ダンジョン探索。
  2. 敵と接触すると戦闘が発生（英単語のスペリングクイズ）。
  3. クイズに正解して敵を撃破し、経験値とゴールドを獲得。
  4. レベルアップや強力な装備を入手しながら地下5階からの生還（脱出）を目指す。

---

## 2. 探索システム
* **ダンジョンマップ:**
  - 各フロアは縦20マス × 横20マスのグリッド。
  - フロア移動時に**自動ランダム生成（部屋と通路の連結アルゴリズム）**されます。
  - キャラクター・タイル表示記号（アスキーアート）:
    - `@` : プレイヤー（自分）
    - `#` : 壁（通行不可）
    - `.` : 床（通路・部屋）
    - `>` : 階段（次のフロアに進む。踏むと最大HPの25%が自動回復。地下5階の階段を踏むとゲームクリア）
* **ターン制ルール:**
  - プレイヤーが1歩移動（またはSpaceキーで待機）すると、敵も1ターン行動（追跡や攻撃）します。
  - 敵との距離が5マス以内になると、敵はプレイヤーを追跡（最短経路）し始めます。

---

## 3. 戦闘システム（タイピング・クイズ）
* **戦闘開始:** プレイヤーが敵キャラクターのマスに移動すると、エンカウントが発生。
* **バトルクイズ:**
  - 画面が一時停止し、バトル用の入力ボックスが表示されます。
  - 日本語（和訳意味）が表示され、その英単語のスペルを入力して `Enter` キーで決定します。
  - **ヒント機能:** 「ヒントを表示」ボタンを押すことで、文字数に応じた伏字ヒントを表示可能（例: "chest" に対し "c _ _ _ t" など）。最大2段階。
* **ターン解決ロジック:**
  - **正解の場合:** 
    - プレイヤーが敵に攻撃します。与えるダメージ = `(プレイヤーATK) - (敵DEF)` （最低1ダメージ）。
    - 攻撃された敵は「ひるみ（スタン）」状態になり、**そのターンの反撃（敵の攻撃）を行いません**。
  - **不正解の場合:**
    - 攻撃は不発（ミス）となり、敵を攻撃できません。
  - **敵のターン（反撃）:**
    - クイズが終了すると（正解で敵がスタンしている場合、または倒した場合を除く）、プレイヤーと隣接（距離1）しているすべての敵がプレイヤーに反撃を行います。
    - 受けるダメージ = `(敵ATK) - (プレイヤーDEF)` （最低1ダメージ）。
* **戦闘終了と報酬:**
  - 敵のHPが0以下になると消滅。
  - プレイヤーは敵固有の経験値（XP）とゴールド（G）を獲得。
  - **レベルアップ:** 獲得XPが `次のレベルに必要なXP` に達するとレベルアップ：
    - 最大HPが8上昇し、HPが全回復。
    - ATKが2、DEFが1上昇。
    - 必要XPが1.5倍にスケーリング。

## 4. プレイヤーのステータスと装備
* **基本ステータス:**
  - **HP (Health Points):** 現在値 / 最大値。0になるとゲームオーバー。
  - **XP (Experience Points):** 現在値 / レベルアップに必要な値。
  - **LEVEL:** 現在の強さ。
  - **GOLD:** スコア（ハイスコア計算用）。
  - **ATK (攻撃力):** ダメージ量に直結。
  - **DEF (防御力):** 受けるダメージを軽減。
* **装備品 (Items):**
  - グリッド上に落ちており、踏むと自動的にスロットに装備（上書き）されます。
  - 🗡️ **武器スロット (鉄の剣):** 攻撃力 ATK +4。
  - 🛡️ **防具スロット (鉄の盾):** 防御力 DEF +2。

## 5. 登場キャラクターと数値パラメータ

### プレイヤー初期値
* HP: 40/40, ATK: 7, DEF: 2, Level: 1, XP: 0 (必要XP: 20)

### 敵キャラクター（初期値）
※階層が深くなるにつれ、ステータスは `1 + (floor - 1) * 0.25` の倍率で強化されます。
1. **スライム (`s`):** HP: 8, ATK: 3, DEF: 0 (出現: Floor 1〜)
2. **コウモリ (`b`):** HP: 6, ATK: 4, DEF: 1 (出現: Floor 1〜)
3. **スケルトン (`S`):** HP: 15, ATK: 6, DEF: 2 (出現: Floor 2〜)
4. **ゴースト (`G`):** HP: 12, ATK: 5, DEF: 3 (出現: Floor 3〜)

### 出現アイテム
1. **ポーション (`P`):** HPを15回復。
2. **宝箱 (`C`):** 10〜25ゴールドを獲得。
3. **鉄の剣 (`W`):** 武器（ATK +4）。
4. **鉄の盾 (`D`):** 防具（DEF +2）。

## 6. 単語帳 & CSVカスタム機能
* **英単語の自動選定:**
  - フロア階層に応じた難易度の英単語が自動で抽選されます。
  - クイズで間違えた単語は「要復習」フラグが立ち、次回の戦闘で優先的に再出題されます。
* **学習履歴の記録:**
  - 戦闘で遭遇したすべての英単語の「意味」「正解回数」「不正解回数」「復習/マスター Badge」が単語帳画面に記録され、検索・フィルタリング可能です。
* **CSVファイルのロード:**
  - 英語と日本語の意味からなるCSVをアップロードして、カスタム単語リストとしてゲームに読み込ませることができます。
  - ヘッダーあり（`word`,`meaning`）およびヘッダーなしの2列形式両方に対応。

## 7. UIデザイン（デスクトップ・ウィンドウシステム）
* **テーマ:** CRTモニター風ネオングリーン・レトロターミナル風スタイル。
* **マルチウィンドウ機能 (Desktop-OS Style):**
  - デスクトップ環境（画面幅768px以上）において、すべてのゲーム構成要素が**個別の独立したウィンドウ**として動作します。
    - **🗺️ MAP:** ダンジョン画面（キーボード操作補助やD-padトグルボタン含む）
    - **📊 STATUS:** HP/XPバー、能力値、装備中のアイテム
    - **📜 LOGS:** リアルタイムの戦闘・拾得ログ
    - **🔑 LEGEND:** アスキー文字の凡例ガイド
    - **📖 WORDLIST:** 学習ログ・CSVインポート機能
  - **ドラッグ移動:** 各ウィンドウのヘッダー部分をつかんで、デスクトップ領域の任意の場所にドラッグできます。
  - **リサイズ:** 各ウィンドウの右下コーナー（斜線のついたつまみ）をドラッグして大きさを可変できます。
  - **最前面表示:** ウィンドウをクリックすると `zIndex` が動的に上昇し、最前面に表示されます。
  - **位置の永続化:** レイアウト位置とサイズは `localStorage` に即座に保存され、リロードしても配置が維持されます。
  - **リセットボタン:** `🔄 RESET` ボタンでデフォルトの3列風配置にいつでもリセット可能。
  - **モバイル対応:** 画面幅768px未満では、ウィンドウ機能が無効化され、スクロール可能な縦並びのシンプル配置に自動変換されます。

## 8. ソースコードファイル構成
* `src/main.jsx`: エントリーポイント。StrictModeとCSS読み込み。
* `src/index.css`: Viewport全体のスクロールロック、html/body/rootのサイズ定義。
* `src/App.jsx`: ゲームの主ロジック、ターン進行、ウィンドウマネージャー（ドラッグ・リサイズ・重なり制御等）、Dungeons生成処理。
* `src/App.css`: デスクトップ領域、各フローティングウィンドウ、D-pad、HPバー、レトロフォント等の基本スタイル。
* `src/components/TileMap.jsx` / `.css`: マップタイルの2Dグリッド描画コンポーネント。
* `src/components/Tile.jsx` / `.css`: 単一タイルの表示処理。プレイヤー、敵、床などのカラーリング定義。
* `src/components/QuizOverlay.jsx` / `.css`: 戦闘タイピング入力、スペルチェック、伏字ヒント表示。
* `src/components/WordListPanel.jsx` / `.css`: PapaParseによるCSVロード、学習統計表示、単語テーブル。
* `src/utils/words.js`: 難易度別（Floor 1〜5）のデフォルト英単語データベース、および復習単語を優先するランダム抽出アルゴリズム。
* `src/utils/sound.js`: Web Audio API を使用し、オーディオファイルを読み込むことなく、オシレーター（シンセ）でレトロな移動音・被ダメージ音・勝利音などの効果音を生成。
